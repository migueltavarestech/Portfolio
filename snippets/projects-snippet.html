migueltavarestech/health-check
LANGUAGE USED
Python
MODULES USED
pandas
requests
time
cloudscraper
urllib3
BeautifulSoup
PURPOSE
Determine the connection status of any list of websites given on a .csv file, by specifying if the website is Live, Dead, IP Blocked or Protected by Captcha
METHODOLOGY
-Transform .csv into a pd dataframe and iterate through list of websites
-Start a requests session and record status_code
-Scrape for "Access Denied" or "IP Blocked" words
-Determine if website is blank inside
-Scrape for captcha forms
-Output .csv file with websites and final results
